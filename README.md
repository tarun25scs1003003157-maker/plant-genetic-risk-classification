# 🌱 Plant Genetic Risk Classification

## 📌 Project Overview

This project uses Machine Learning to classify plant genetic data into **Safe** or **Risky** categories.

## ⚙️ Technologies Used

* Python
* Pandas
* Matplotlib
* Scikit-learn

## 🤖 ML Algorithms

* Decision Tree Classifier
* K-Means Clustering

## 📊 Features Used

* Gene Length
* Mutation Rate
* Trait Strength

## 📈 Outputs

* Accuracy Score
* Confusion Matrix
* Bar Graph
* Clustering Visualization

## 🚀 How to Run

```bash
pip install pandas matplotlib scikit-learn
python project.py
```

## 🧪 Sample Prediction

The model can predict whether a new plant is **Safe** or **Risky** based on input data.
